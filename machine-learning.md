# Machine Learning in Bioinformatics

Machine learning (ML) algorithms allow us to predict biological properties and discover patterns in large datasets.

## Introduction to ML

*   **Training vs. Testing**: Splitting data to evaluate performance.
*   **Overfitting vs. Underfitting**: Balancing model complexity.
*   **Cross-Validation**: K-fold cross-validation.

## Supervised Learning

Used for classification (discrete labels) and regression (continuous values).

*   **Algorithms**:
    *   Support Vector Machines (SVM)
    *   Random Forests
    *   Logistic Regression
    *   K-Nearest Neighbors (KNN)
*   **Applications**: Disease diagnosis, protein structure prediction.

## Unsupervised Learning

Used for discovering structure in unlabeled data.

*   **Clustering**: K-means, Hierarchical clustering (Heatmaps).
*   **Dimensionality Reduction**: Principal Component Analysis (PCA), t-SNE, UMAP.
*   **Applications**: Cell type identification in scRNA-seq, patient stratification.

## Deep Learning

*   **Neural Networks**: CNNs (for sequence motifs), RNNs.
*   **Applications**: AlphaFold (protein folding), variant effect prediction.
